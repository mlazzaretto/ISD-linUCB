# Invariance-based dynamic regret minimization
This repository contains the code for the simulation experiments presented in the paper 'Invariance-based dynamic regret minimization'

The code in the 'isd' folder is by Lazzaretto, Margherita, Jonas Peters, and Niklas Pfister. "Invariant Subspace Decomposition." Journal of Machine Learning Research 26.95 (2025): 1-56. https://github.com/mlazzaretto/Invariant-Subspace-Decomposition


### Experiments files
Section 5 -- Simulation experiments:

    5.1 Oracle subspace decomposition: test_oracle_ISD_pinv.py, test_oracle_pres.py

    5.2 Estimated subspace decomposition: test_ISD_T0.py

Section B3 -- Varying parameter within the evaluation window

    test_oracle_ISD_pinv_varying.py


### Instructions:
Python version used: 3.8.5

All dependencies are specified through COMMANDFORLOADPACKAGE in the individual files.
